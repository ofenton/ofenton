# Running Notebooks Help

To run the test_basemap.ipynb file either

* On command-line
```
workon mapping
ipython
run test_basemap.ipynb
deactivate
```

OR

* For a Browser
```
workon mapping
ipython notebook test_basemap.ipynb
deactivate
```

  * Note: You might need to add the following to your notebook 
```%pylab inline```
